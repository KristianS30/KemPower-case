# Prototype scope

## Prototype description

The prototype has 20 light bulbs, that can be controlled with a software, the controller. You can control any light bulb on its own or create a preset that can then be applied to all the bulbs you have selected. You can also create a preset category, and when you assign a category to a group of bulbs, you can control them all simultaneously without having to select the bulbs themselves.
In the controller, you can name the bulbs based on their location and function. This allows you to check if they are configured in a way you want.
