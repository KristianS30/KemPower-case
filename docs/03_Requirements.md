# Requirements for RoomLight

| REQ_ID | Requirement                                                        | Type           | Priority |
| ------ | ------------------------------------------------------------------ | -------------- | -------- |
| 001    | The controller connects wirelessly to all the lights in the hotel  | Functional     | Must     |
| 002    | You can apply settings to multiple lights simultanously            | Functional     | Must     |
| 003    | You can create new light presets                                   | Functional     | Must     |
| 004    | One light lasts at least 25000 hours                               | Non-functional | Must     |
| 005    | The contoller will raise an alert, if a light goes out             | Functional     | Should   |
| 006    | You can create preset categories for different room configurations | Functional     | Must     |
| 007    | You can control the brightness and color of the lights             | Functional     | Must     |
| 008    | Light bulbs are available in all the standard base sizes           | Non-functional | Should   |
| 009    | There are bulbs available for both inside and outside use          | Non-functional | Should   |
| 010    | You can name the lights in the controller for ease of use          | Functional     | Must     |
| 011    | You can see how many hours a bulb has been used                    | Functional     | Should   |
| 012    | Only the owner of the controller can connect to the lights         | Non-functional | Must     |
| 013    | The controller is locked with a password                           | Functional     | Must     |
